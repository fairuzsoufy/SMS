INSTRUCTIONS

1. Create a new database and name it sms_master 
2. Import the .sql file under SMS/database/MySQL
3. Fork/download this repo and extract it to a folder titled "SMS" under your XAMPP/htdocs directory
4. Use the "php artisan serve" command in your command prompt (or terminal on Mac) with Apache and MySQL running (XAMPP)
5. Visit 127.0.0.1:8000 in your preferred browser
